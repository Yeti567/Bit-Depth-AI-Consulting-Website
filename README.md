# BitDepth AI Consulting Ltd. Website

Marketing website built with Next.js, TypeScript, and Tailwind CSS.

## Included pages

- Home
- Services
- Industries
- Industries / Oil & Gas
- Industries / Manufacturing
- AI Audit
- About
- Resources
- Privacy Policy

## Development

1. Install dependencies
2. Start the dev server

Recommended commands:

```bash
npm install
npm run dev
```

## Notes

- Existing project imagery is used from `public/images`
- Shared content is centralized in `src/lib/site-data.ts`
- Reusable layout and UI components live in `src/components`
- The AI Audit form is a placeholder capture form ready to be connected to a CRM, email workflow, or booking embed
